# wefunk_pl
Python script to download playlist info from wefunkradio.com into JSON format

WEFUNK is a Hip Hop, Funk & Soul mix show broadcasting from Montreal. Tune in and enjoy underground hip hop, classic funk and rare grooves.

Prerequisites
-------------

* Python
* BeautifulSoup library

Quickstart
--------------

```
pip install requests beautifulsoup4
```

```
python wefunkpl.py
```
enter show# at prompt
